# README_REFACTOR_PACKAGE.md

Пакет содержит предложенный рефакторинг документации для ветки `docs_branch/docs`.

## Что внутри

- обновлённый `01_project_foundation.md` как более тонкий foundation-документ;
- согласованные child docs `02-14`;
- перенос детализации из `01` в профильные документы;
- единое правило: foundation = что и почему, child docs = как именно.

## Как использовать

1. Распаковать архив.
2. Сравнить содержимое с текущими файлами репозитория.
3. Заливать в Git по частям:
   - сначала `01`, `05`, `06`, `08`, `09`, `10`, `11`;
   - потом `12`, `14`;
   - затем scaffolding `02`, `03`, `04`, `13`.
